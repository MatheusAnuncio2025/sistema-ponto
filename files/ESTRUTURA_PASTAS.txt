LEGENDA:
========
(a criar) = Arquivos/pastas que serão criados nas próximas fases
└── = Estrutura de árvore


COMO CRIAR ESTA ESTRUTURA MANUALMENTE:
=======================================

# Na raiz do seu projeto:
mkdir sistema-ponto
cd sistema-ponto

# Criar estrutura de documentação
mkdir -p docs/api docs/user-manual docs/admin-manual

# Criar estrutura do backend
mkdir -p backend/src/{config,controllers,models,routes,middlewares,services,utils,jobs}
mkdir -p backend/{migrations,seeders,tests}

# Criar estrutura do frontend  
mkdir -p frontend/src/{components,pages,contexts,services,utils,types,styles}
mkdir -p frontend/public

# Depois é só colocar os arquivos que vou te enviar em cada pasta!
