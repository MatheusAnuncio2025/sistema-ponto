INSTRUÇÕES RÁPIDAS
==================

OPÇÃO 1: MONTAGEM AUTOMÁTICA (RECOMENDADO)
-------------------------------------------

Mac/Linux:
1. Baixe todos os 25 arquivos para uma pasta
2. Abra o terminal nessa pasta
3. Execute: chmod +x montar-projeto.sh
4. Execute: ./montar-projeto.sh
5. Pronto! Estrutura montada automaticamente

Windows:
1. Baixe todos os 25 arquivos para uma pasta
2. Clique com botão direito no arquivo "montar-projeto.ps1"
3. Selecione "Executar com PowerShell"
4. Pronto! Estrutura montada automaticamente

COMO USAR NO GITHUB
===================

1. Monte a estrutura (manual ou com script)
2. Inicialize o Git:
   git init
   git add .
   git commit -m "Inicial commit: Estrutura base"
   
3. Crie repositório no GitHub (github.com/new)

4. Conecte e envie:
   git remote add origin [URL_DO_REPOSITORIO]
   git branch -M main
   git push -u origin main

5. Em outra máquina:
   git clone [URL_DO_REPOSITORIO]
   cd sistema-ponto
   # Siga INICIO-RAPIDO.md
